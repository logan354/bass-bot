# Bass
A music bot for Discord. With support for YouTube, Spotify and Soundcloud!


## Features

- ✅ Build on Discord.js v13
- ✅ Supports YouTube, Spotify and Soundcloud!

## Commands

Bass has 4 main command categories

- **Track**: `disconnect`, `forceskip`, `join`, `loop`, `nowplaying`, `pause`, `play`, `resume`, `search`, `seek`, `voteskip`
- **Queue**: `clear`, `loopqueue`, `queue`, `shuffle`
- **Premium**: `volume`
- **Utility**: `help`, `invite`, `ping`, `statistics`
